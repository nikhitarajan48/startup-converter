# 🚀 Problem → Startup Converter

A web app that converts real-world problems into structured startup ideas using **Lean Startup methodology**, powered by **Groq AI (Free)**.

[Click here to open the app](#) ← Streamlit link goes here after deploy

## ⚙️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🚀 Deploy on Streamlit Cloud (Free)
1. Push this folder to GitHub
2. Go to share.streamlit.io
3. Connect your GitHub repo
4. Set `app.py` as the main file
5. Done — get a free public link!

## Tech Stack
- Python + Streamlit
- Groq API (Llama 3.3 70B) — Free
- Lean Startup Methodology
